## 🏆 Featured Projects

### 💉 [Safety-Constrained Insulin Dosing RL Agent](https://github.com/swarat17/insulin-rl-agent)
Custom Gymnasium environment over an FDA-validated T1D simulator benchmarking safety-constrained RL agents for autonomous insulin dosing.
- **Tech Stack:** PyTorch, Stable-Baselines3, Gymnasium, Simglucose, Streamlit, Hugging Face
- **Results:** 76.5% safe-range adherence matching clinician baseline; recovered collapsed PPO policy from 47% → 73% time-in-range via Lagrangian safety constraints
- **Scale:** 5 agents (DQN, PPO, SAC + constrained variants) benchmarked across 3 patient cohorts
- [Repository](https://github.com/swarat17/insulin-rl-agent)

### 🔬 [Research Synthesis & Hypothesis Agent](https://github.com/swarat17/research-synthesis-agent)
Autonomous 7-node multi-agent LangGraph pipeline that fetches ArXiv papers, synthesizes findings, detects contradictions, and generates novel research hypotheses.
- **Tech Stack:** GPT-4o-mini, Claude Sonnet, LangGraph, Pinecone, FastAPI, Streamlit, AWS Lambda, Supabase
- **Performance:** ~$0.035/query for 10 papers with cost-aware GPT-4o/Claude routing and configurable spending caps
- **Output:** 400–600 word synthesis + 3 scored hypotheses with contradiction severity ratings; serverless AWS Lambda deployment
- [Repository](https://github.com/swarat17/research-synthesis-agent)

### 🧬 [FBS Design Context AI](https://github.com/swarat17/fbs-design-ai)
AI-assisted design decomposition system implementing the Function-Behavior-Structure ontology with fine-tuned LLMs, validating that task-specific models outperform general-purpose baselines.
- **Tech Stack:** PyTorch, Flan-T5-Small, Gradio, Fine-tuning, Gemini API
- **Results:** 92% average ROUGE-L improvement across 3 task-specific models; 127% improvement on Function→Behavior task
- **Innovation:** 3 specialized Flan-T5-Small models (R→F, F→Be, Be→S) trained on ~400 examples each on consumer hardware (GTX 1650)
- [Repository](https://github.com/swarat17/fbs-design-ai)

### 🤖 [RL-Guided Curriculum Learning Pipeline](https://github.com/swarat17/rl-driven-text-curriculum)
Novel approach to curriculum learning using a PPO agent for adaptive LLM-based text classification training.
- **Tech Stack:** PyTorch, Hugging Face Transformers, Gymnasium, BERT, PPO
- **Results:** ~60% faster convergence through online difficulty estimation and easy-to-hard data ordering
- **Innovation:** Custom Gymnasium environment with composite reward functions and BERT embedding analysis
- [Repository](https://github.com/swarat17/rl-driven-text-curriculum)
